What's this aboot, eh?

This package will allow you to make a site for your Django web app!
It provides a skeleton to link the static pages of your app together...
You can link these to other pages in your app: user pages, input pages, etc.

A list of views in this skeleton:
  • An index page
  • An about page
  • An FAQ page
  • A contact form

Use the menu bar at the top to navigate to these pages.
